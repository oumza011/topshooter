# Top-Down Shooter

Godot 4 prototype for a sci-fi escort shooter demo.

## Demo

Open `project.godot` in Godot and press Play.

Goal: protect Mila and escort her to the green rescue beacon.

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left mouse button
- Restart after win/fail: `R`
- Open art reference gallery: `Esc`

## Art References

Generated concept references are stored in `art/reference`.
